### note 
Kompressor Image Secara otomatis membuat 5 ukuran image dan memasukannya kedalam jenisnya masing-masing.
1. thumbnail (100x100)
2. small (150x150)
3. medium (400x400)
4. tablet (1000x1000)
5. large (1900x1900) 

jika ingin mencobanya, tinggal tambahkan saja image baru di folder source.


## Installing
npm install

## Start / Running 
npm start
